# MSN Chat log parser

This parser uses Beautiful Soup to convert MSN Messenger chat logs from XML to a more readable text file.

## Usage

`pipenv run python parser.py --input-folder=<input_folder> --output-folder=<output_folder> --print-output`

### To run tests

`pipenv run pytest`
